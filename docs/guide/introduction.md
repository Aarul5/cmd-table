# Introduction

**cmd-table** is a modern, feature-rich, and enterprise-grade CLI table library for Node.js.

## Why cmd-table?

- **Zero Dependencies**: Lightweight and fast.
- **Modern Theming**: "Rounded" theme by default, plus many others.
- **Rich Styling**: ANSI color support for headers and columns.
- **Advanced Layouts**: `colSpan`, `rowSpan`, auto-sizing, word-wrap.
- **Responsive**: Hide or stack columns on smaller screens.
- **Interactive TUI**: Built-in interactive mode for exploring large datasets.
- **Data Visualization**: Sparklines, Heatmaps, Tree Views.
- **Integrations**: React/JSX, HTML, CSV, JSON.
- **Streaming**: Efficiently render large datasets.
